# Words from word

It's HTML-5 game about creating different words from one big word for Ya-Games(https://yandex.ru/games).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
